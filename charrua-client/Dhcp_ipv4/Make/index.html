Connect to an ipv4 device using information from a DHCP lease. If cidr is provided, no DHCP requests will be done, but instead a static IPv4 (Tcpip.Ip.S) stack will be used. If no_init is provided and true, nothing will be initialized (for dual IPv4 and IPv6 stack where only the IPv6 part should be used).
